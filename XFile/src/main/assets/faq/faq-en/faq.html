How to "share" content?

You can share any types of files of any sizes without using mobile data

3 options to choose from:

  1. Press and hold the file icon, wait until the icon is enlarged, then slightly flick the phone
  2. Press and hold the file icon, wait until the icon is enlarged, drag and drop to the recipient avatar
  3. Use one finger to press and hold the recipient avatar, wait until the avatar is enlarged, then use other finger to repeated click the files to be shared.

Tip:

Press and hold the file icon, then release the icon to enter multi-selection mode
